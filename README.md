#  Calculadora React Native

Calculadora simples e funcional desenvolvida em React Native.

##  Funcionalidades

-  Adição
-  Subtração
-  Multiplicação
-  Divisão
-  Números decimais
-  Limpar (C)
- Interface moderna e responsiva

## Tecnologias

- React Native
- Expo
- JavaScript

## Estrutura do Projeto

calculadora/
├── App.js          # Componente principal
├── Display.js      # Componente do display
├── Teclado.js      # Componente do teclado
├── Botao.js        # Componente de botão reutilizável
└── styles.js       # Estilos da aplicação
```
